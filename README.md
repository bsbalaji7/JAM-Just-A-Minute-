JAM – Just A Minute (Quick Commerce Platform)

JAM – Just A Minute is a full-stack Quick Commerce web application designed to deliver groceries, food, electronics, and daily essentials to customers within minutes. The platform provides a fast, simple, and modern online shopping experience, allowing users to browse products, add them to a cart, and place orders quickly.

This project demonstrates the implementation of a complete e-commerce system using modern web technologies and follows the MVC architecture for clean project structure and scalability.

📌 Project Overview

JAM aims to simulate a real-world quick commerce platform similar to modern instant delivery services. The platform focuses on speed, usability, and efficiency, enabling customers to purchase everyday items quickly without visiting physical stores.

The system includes user authentication, product management, shopping cart functionality, and order processing, making it a fully functional online marketplace.

⚙️ How the System Works

User registers or logs into the system.

The user browses products from different categories.

Products can be added to the shopping cart.

The user proceeds to checkout.

Order details are stored in the MongoDB database.

Users can view their order history anytime.
